Game Rules

  • Select the lock pin buttons in a sequnece to unlock the door.
  • The pin will display green if its correct and red if its wrong.
  • For every incorect sequence you will get a wrong guess added to the counter,
  • If you select the wrong button you can clear them by pressing reset.
  • Once all pins are correct press unlock to move onto the next level.
  • Good luck!
